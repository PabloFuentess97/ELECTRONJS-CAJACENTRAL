# ELECTRONJS-CAJACENTRAL
Crear .env
````
# Configuración de rutas
# --  Es la URL donde se encuentra la API REST que maneja los registros -- #
APP_RUTE_APIURL=http://localhost:3000/registros
# -- Puerto en el que se encuentra escuchando la API REST -- #
APP_RUTE_APIPORT=3000
# -- Es la ruta donde se encuentra la carpeta donde se almacenan los archivos JSON con los registros de entrada. Es una carpeta externa a la aplicación -- #
APP_RUTE_JSONFOLDER=C:/Users/34622/Documents/BaseDatosJSON
# -- Es la ruta donde se encuentra la carpeta que contiene las imágenes de las matrículas que se van a procesar -- #
APP_RUTE_MATRICULASFOLDER=C:/Users/34622/Documents/Matriculas
# -- Es la ruta donde se encuentra el archivo de texto que contiene las matrículas que han sido bloqueadas y sus respectivos motivos -- #
APP_RUTE_BLACKLISTFILE=C:/Users/34622/Documents/listanegra.txt
````
